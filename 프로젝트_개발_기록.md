### 파일별 역할

| 파일 | 라인 수 | 주요 기능 |
|------|---------|----------|
| `index.html` | ~400 | 히어로 섹션, 멤버 플립 카드, 통계 카운터 |

| `library.js` | ~300 | 검색, 필터, 정렬, 즐겨찾기 기능 |
| `book-detail.html` | ~600 | 리뷰 작성, 별점, 답글 시스템 |

| `community.html` | ~200 | 게시판, 공지사항, 독서 명언 |

## 🎯 배운 점 & 팁

### AI 협업 시 효과적인 프롬프트 작성법

1. **구체적으로**: "버튼 추가해줘" → "헤더 오른쪽에 파란색 로그인 버튼 추가해줘"
2. **조건 명시**: "반응형으로 해줘" → "768px 이하에서는 세로 배치로 해줘"
3. **현재 상태 설명**: "안 돼" → "iOS Safari에서 플립 애니메이션이 안 돼"
4. **한 번에 결정**: 여러 옵션 중 고민되면 AI에게 먼저 물어보기
